# What
Pixelboxer takes in a ppm file, puts a border around every pixel after scaling it to the inputed blow-up factor.

# How
```
pixelboxer <input-ppm-file> <output-ppm-file> <blow-up-factor>

Eg: pixelboxer in.ppm out.ppm 10
```
